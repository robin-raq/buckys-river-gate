What changes in the actual rebuild

  • Delete the current RiverScene.tsx, river-scene.css, the 28% padding hack, and the layered scene-bg.png.
  • Background: a single CSS rule on .lesson-main — background: url('/bucky-background.png') center / cover no-repeat.
  • Lane grid: drop our DOM-rendered SnapGuides. The dashed lines are pixels in the PNG.
  • Bucky sprite: position: absolute; bottom: 18%; left: 5%; width: 13%. The existing BuckyBeaverArt already handles state changes.
  • Speech bubble: anchored upper-left in the sky band with the tail pointing down-left toward Bucky.
  • Reference gate: horizontal cyan-glow bar spanning x: 14% → 86% of the river area.
  • Dock-tray: position absolute at bottom: 4%; left: 36%; right: 36% so logs sit on the wooden dock.
  • Goal sidebar: small wooden card on the right bank, bottom: 30%; right: 2%.
  • Top bar: wooden plank gradient pulling from the dock palette.
