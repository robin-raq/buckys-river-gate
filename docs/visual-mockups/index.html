Design system (from GDD)

  • Log width = fraction — size is the math; color reinforces (amber → gold → pine).
  • Reference Gate — pulsing blue target the child matches in the build zone.
  • Errors — red dashed gap (too short); soft, never punitive.
  • Bucky zone — avatar + bubble + phase progress dots.
  • Chop flow — long-press → dashed saw line + amber pulse → tap confirm → split + chips; 1/4 wiggles + lock if too small.
  • 1/1 chop — whole log splits into 2× 1/2 (tabs Chop 1/1, 1/1 → halves).
  • Storyboard — see CHOP_STORYBOARD.md for frame timing handoff.
